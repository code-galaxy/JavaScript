Task 5

Task 7

Task 8

Task 10

На странице создан div.out-10. С помощью innerHTML запишите внутрь него следующую строку '<h2>Hi</h2>'.

Task 11

На странице создан div.out-11. Запишите внутрь него строку '123'. Для записи используйте innerHTML. Второй операцией - допишите в div.out-11 строку '456'. Для дозаписи примените +=.

Обратите внимание - код уже набран, вам нужно удалить комментарий и изучить принцип работы.

Task 12

На странице создан div.out-12. Получите его в переменную a. С помощью innerHTML запишите внутрь переменной a число 3.1415. Изучите как в JS пишутся дроби - через точку или через запятую!

Task 13

На странице создан div.out-13. Получите его в переменную out13.

С помощью innerHTML запишите в него строку:

'<img src="https://itgid.info/img/js20/js20_unit_01.png" alt="js">'

Обратите внимание. Вам придется вставить строку, которая уже содержит кавычки. Главное чтобы кавычки чередовались.

Поэтому при вставке оберните данную строку одинарными кавычками. Если все сделано верно - то на странице появится картинка.

Здесь будет изображение.
